## 📌 About the Project

PomoBee is a full-stack productivity application based on the Pomodoro Technique, designed to help users improve focus, manage time efficiently, and track productivity.

This project was built with a strong emphasis on real-world engineering practices including system design, scalability thinking, and deployment.

---

## ⚙️ Features

* ⏱️ Pomodoro Timer (25 min focus, short & long breaks)
* 🔐 User Authentication (JWT-based login/register)
* 📊 Session Tracking & History
* ⚡ Performance Optimization using Caching
* 🌐 Fully Deployed (Frontend + Backend)
* 📱 Responsive UI
* 📄 Legal Pages (Privacy Policy, Terms, Contact)

---

## 🧠 Architecture & Design

The application follows a layered architecture:

Frontend (React)
↓
Backend (Spring Boot REST APIs)
↓
Cache Layer (Spring Cache)
↓
Database (MongoDB Atlas)

### Key Design Decisions:

* Separation of concerns using Controller → Service → Repository pattern
* Use of JWT for stateless authentication
* Caching to reduce database calls and improve performance
* Designed to handle backend cold starts (Render free tier)
